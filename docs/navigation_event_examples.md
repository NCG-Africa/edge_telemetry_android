## Activity Navigation

### Example 1: App Launch (First Screen)

**Scenario**: User opens app, MainActivity is launched

**Code**:
```kotlin
class MainActivity : AppCompatActivity() {
    override fun onCreate(savedInstanceState: Bundle?) {
        super.onCreate(savedInstanceState)
        setContentView(R.layout.activity_main)
    }
}
```

**Generated Event**:
```json
{
  "type": "event",
  "eventName": "navigation",
  "timestamp": "2024-03-18T10:00:00.000Z",
  "attributes": {
    "navigation.from_screen": null,
    "navigation.to_screen": "MainActivity",
    "navigation.method": "push",
    "navigation.route_type": "main_flow",
    "navigation.has_arguments": false,
    "navigation.timestamp": "2024-03-18T10:00:00.000Z"
  }
}
```

**Key Points**:
- `from_screen` is `null` on first launch
- `method` is `push` for forward navigation
- `route_type` is `main_flow` for root activities
- `has_arguments` is `false` (no intent extras)

### Example 2: Activity to Activity Navigation

**Scenario**: User navigates from MainActivity to ProfileActivity

**Code**:
```kotlin
// MainActivity
class MainActivity : AppCompatActivity() {
    fun openProfile() {
        val intent = Intent(this, ProfileActivity::class.java)
        intent.putExtra("user_id", "12345")
        startActivity(intent)
    }
}

// ProfileActivity
class ProfileActivity : AppCompatActivity() {
    override fun onCreate(savedInstanceState: Bundle?) {
        super.onCreate(savedInstanceState)
        val userId = intent.getStringExtra("user_id")
    }
}
```

**Generated Event**:
```json
{
  "type": "event",
  "eventName": "navigation",
  "timestamp": "2024-03-18T10:05:30.123Z",
  "attributes": {
    "navigation.from_screen": "MainActivity",
    "navigation.to_screen": "ProfileActivity",
    "navigation.method": "push",
    "navigation.route_type": "main_flow",
    "navigation.has_arguments": true,
    "navigation.timestamp": "2024-03-18T10:05:30.123Z"
  }
}
```

**Key Points**:
- `from_screen` tracks previous activity
- `has_arguments` is `true` (intent has extras)

### Example 3: Activity with Deep Link

**Scenario**: User clicks deep link to open ProductActivity

**Code**:
```kotlin
// AndroidManifest.xml
<activity android:name=".ProductActivity">
    <intent-filter>
        <action android:name="android.intent.action.VIEW" />
        <category android:name="android.intent.category.DEFAULT" />
        <category android:name="android.intent.category.BROWSABLE" />
        <data android:scheme="myapp" android:host="product" />
    </intent-filter>
</activity>

// ProductActivity
class ProductActivity : AppCompatActivity() {
    override fun onCreate(savedInstanceState: Bundle?) {
        super.onCreate(savedInstanceState)
        val productId = intent.data?.getQueryParameter("id")
    }
}
```

**Deep Link**: `myapp://product?id=123`

**Generated Event**:
```json
{
  "type": "event",
  "eventName": "navigation",
  "timestamp": "2024-03-18T10:10:15.456Z",
  "attributes": {
    "navigation.from_screen": null,
    "navigation.to_screen": "ProductActivity",
    "navigation.method": "push",
    "navigation.route_type": "deeplink",
    "navigation.has_arguments": true,
    "navigation.timestamp": "2024-03-18T10:10:15.456Z"
  }
}
```

**Key Points**:
- `route_type` is `deeplink` (auto-detected from intent.data)
- `from_screen` is `null` (external entry)
- `has_arguments` is `true` (deep link has data)

## Fragment Navigation

### Example 4: Fragment Transaction

**Scenario**: User navigates from HomeFragment to SettingsFragment

**Code**:
```kotlin
class MainActivity : AppCompatActivity() {
    fun navigateToSettings() {
        val fragment = SettingsFragment().apply {
            arguments = Bundle().apply {
                putString("section", "notifications")
            }
        }
        
        supportFragmentManager.beginTransaction()
            .replace(R.id.fragment_container, fragment)
            .addToBackStack("settings")
            .commit()
    }
}

class SettingsFragment : Fragment() {
    override fun onResume() {
        super.onResume()
        val section = arguments?.getString("section")
    }
}
```

**Generated Event**:
```json
{
  "type": "event",
  "eventName": "navigation",
  "timestamp": "2024-03-18T10:15:20.789Z",
  "attributes": {
    "navigation.from_screen": "HomeFragment",
    "navigation.to_screen": "SettingsFragment",
    "navigation.method": "push",
    "navigation.route_type": "fragment_flow",
    "navigation.has_arguments": true,
    "navigation.timestamp": "2024-03-18T10:15:20.789Z"
  }
}
```

**Key Points**:
- `route_type` is `fragment_flow` for all fragments
- `has_arguments` is `true` (fragment has arguments bundle)

## Jetpack Compose Navigation

### Example 5: Compose Screen Navigation

**Scenario**: User navigates from Home to Profile screen in Compose

**Code**:
```kotlin
@Composable
fun MyApp() {
    val navController = rememberNavController()
    
    NavHost(navController = navController, startDestination = "home") {
        composable("home") {
            TrackComposeScreen(navController = navController, screenName = "HomeScreen")
            HomeScreen(onNavigateToProfile = { 
                navController.navigate("profile")
            })
        }
        
        composable("profile") {
            TrackComposeScreen(navController = navController, screenName = "ProfileScreen")
            ProfileScreen()
        }
    }
}
```

**Generated Event** (Profile screen):
```json
{
  "type": "event",
  "eventName": "navigation",
  "timestamp": "2024-03-18T10:20:45.234Z",
  "attributes": {
    "navigation.from_screen": "HomeScreen",
    "navigation.to_screen": "ProfileScreen",
    "navigation.method": "push",
    "navigation.route_type": "main_flow",
    "navigation.has_arguments": false,
    "navigation.timestamp": "2024-03-18T10:20:45.234Z"
  }
}
```

### Example 6: Compose Navigation with Arguments

**Scenario**: Navigate to product detail with product ID

**Code**:
```kotlin
NavHost(navController = navController, startDestination = "products") {
    composable("products") {
        TrackComposeScreen(navController = navController, screenName = "ProductListScreen")
        ProductListScreen(onProductClick = { productId ->
            navController.navigate("product/$productId")
        })
    }
    
    composable(
        route = "product/{productId}",
        arguments = listOf(navArgument("productId") { type = NavType.StringType })
    ) { backStackEntry ->
        val productId = backStackEntry.arguments?.getString("productId")
        TrackComposeScreen(navController = navController, screenName = "ProductDetailScreen")
        ProductDetailScreen(productId = productId)
    }
}
```

**Generated Event**:
```json
{
  "type": "event",
  "eventName": "navigation",
  "timestamp": "2024-03-18T10:25:12.567Z",
  "attributes": {
    "navigation.from_screen": "ProductListScreen",
    "navigation.to_screen": "ProductDetailScreen",
    "navigation.method": "push",
    "navigation.route_type": "main_flow",
    "navigation.has_arguments": true,
    "navigation.timestamp": "2024-03-18T10:25:12.567Z"
  }
}
```

**Key Points**:
- `has_arguments` is `true` (route has productId parameter)

## Field Reference

### Required Fields

| Field | Type | Description | Example |
|-------|------|-------------|---------|
| `navigation.to_screen` | String | Destination screen name | `"ProfileScreen"` |
| `navigation.method` | String | Navigation action | `"push"`, `"pop"`, `"replace"` |
| `navigation.timestamp` | String | ISO 8601 timestamp | `"2024-03-18T14:50:23.456Z"` |

### Optional Fields

| Field | Type | Description | Example |
|-------|------|-------------|---------|
| `navigation.from_screen` | String | Source screen (null on launch) | `"HomeScreen"` or `null` |
| `navigation.route_type` | String | Route classification | `"main_flow"`, `"modal"`, `"deeplink"` |
| `navigation.has_arguments` | Boolean | Has navigation data | `true`, `false` |

---

## Navigation Method Reference

| Method | Description | When Used |
|--------|-------------|-----------|
| `push` | Forward navigation | New Activity/Fragment, Compose navigate() |
| `pop` | Back navigation | Back button, popBackStack() |
| `replace` | Screen replacement | Activity finish+start, Fragment replace |

---

## Route Type Reference

| Route Type | Description | Auto-Detected For |
|------------|-------------|-------------------|
| `main_flow` | Primary navigation | Root activities, Compose default |
| `fragment_flow` | Fragment navigation | All fragments |
| `deeplink` | Deep link entry | Activities with intent.data |
| `modal` | Modal screens | Custom via additionalData |
| `onboarding` | Onboarding flow | Custom via additionalData |
| `settings` | Settings screens | Custom via additionalData |
